# ML & AI Capstone Project

## Overview
This repository contains my capstone project for the Professional Certificate in Machine Learning and Artificial Intelligence.

The project aims to apply core machine learning concepts to a real-world problem, covering data exploration, modelling, evaluation and interpretation of results.

## Objectives
- Understand and explore the dataset
- Apply appropriate machine learning techniques
- Evaluate model performance
- Interpret results in a practical, real-world context
